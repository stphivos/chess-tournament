# chess-tournament
Chess tournament single-page application for judge and contestant progress, based on the swiss-system.

## Stack
*   AngularJS 1.4
*   Django REST framework 3.2
*   Django 1.8
*   Python 2.7
*   PostgreSQL 9.4
